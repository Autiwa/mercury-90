
The directory unitary_tests must exist: the data files are there with the gnuplot files allowing to plot.

The directory mercury-original contains the fortran 77 files of the original Mercury code (Chambers 1999)

The directory simu_test is used by the script tests_mercury.py in order to compare the original binary files and the new ones.


-----------------------------------------
Le dossier "unitary_tests" doit exister pour que les tests unitaires puissent se faire.
En effet, c'est là que sont stockés les fichiers de données et les fichiers gnuplot qui permettent de tracer les courbes. 

Le dossier mercury_original contient les fichiers de mercury en version f77 fournis par chambers sur son site internet

Le dossier simu_test est utilisé par le script tests_mercury.py pour comparer les binaires originaux et nouveaux.
